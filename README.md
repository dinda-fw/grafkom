# Visualisasi 3D Objek Kampus (OpenGL)
Proyek ini bertujuan untuk memvisualisasikan beberapa objek arsitektur dan elemen lingkungan kampus menggunakan grafika komputer. Proyek ini mencakup pemodelan objek kantin, gedung fakultas, dan elemen alam seperti danau.

## 📌 Deskripsi Proyek
Proyek ini dikembangkan menggunakan **C++** dan library **OpenGL** untuk merancang objek-objek berikut:
1. **Kantin**: Pemodelan struktur tempat makan dengan detail pada atap dan area terbuka.
2. **Fakultas Hukum**: Representasi gedung fakultas dengan karakteristik arsitektur yang khas.
3. **Danau**: Visualisasi elemen air dan area sekitar untuk menambah estetika lingkungan.

## 🚀 Fitur Utama
- **Pemodelan Geometris**: Menggunakan primitif grafika untuk membentuk struktur bangunan.
- **Transformasi**: Implementasi translasi, rotasi, dan penskalaan agar objek terlihat proporsional.
- **Interaksi Kamera**: Kemampuan untuk melihat objek dari berbagai sudut pandang.

## 🛠️ Teknologi yang Digunakan
- **Bahasa Pemrograman**: C++
- **Graphics Library**: OpenGL (GLUT/GLEW/GLFW)
- **IDE**: Visual Studio
